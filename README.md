Portfolio Web – HTML & CSS
 Descripción del proyecto

Este proyecto consiste en el desarrollo de un portfolio web personal, realizado a partir del template proporcionado en clase y posteriormente personalizado completamente para adaptarlo a mi estilo visual y profesional.

El objetivo principal es mostrar mis trabajos, habilidades y formas de contacto mediante una web moderna, clara y reconocible a nivel estético.

Tecnologías utilizadas

HTML

CSS

Recursos externos para imágenes (ImageKit.io)

Sección Home

Tras implementar el template base, comencé a modificar el diseño empezando por la página principal (Home).

Se añadió un carrusel de imágenes con trabajos propios, cargados mediante ImageKit.io.

El carrusel se integró dentro del encabezado, funcionando como fondo visual de la sección.

Sobre el carrusel se colocó:

Mi nombre

Un tagline

Un botón que redirige a la sección de trabajos

Para lograr este efecto visual, el contenedor del texto y el botón se posicionaron en un nivel superior del eje Z respecto al carrusel.

Diseño y colores

Se modificó la paleta de colores original por una gama personalizada de tonos naranjas y morados, con el objetivo de crear una identidad visual propia y fácilmente reconocible.

Sección “Who is Dalasso?”

En esta sección se incluye:

Una breve descripción personal

Barras de progreso que representan mi nivel de especialización en distintas áreas

Esta parte del sitio sirve para presentar mi perfil profesional de forma visual e intuitiva.

Featured Projects

La sección Featured Projects muestra ejemplos de mis trabajos organizados en tres categorías principales:

Ilustración

Infografía

Edición de imagen y vídeo

Cada categoría se presenta en su propia pestaña, incluyendo:

Una breve descripción

Una imagen representativa del tipo de proyecto

Contacto y redes sociales

La parte final de la página está dedicada al contacto, ofreciendo diferentes formas de comunicación:

Instagram

GitHub

LinkedIn

Correo electrónico personal

Además, el footer incluye un botón que permite volver rápidamente a la sección Home.

Navegación

El sitio cuenta con un menú sticky situado en la parte superior de la web, que:

Permite navegar rápidamente entre secciones

Permanece visible en todo momento mientras se recorre la página

Conclusión

Este proyecto refleja tanto mis habilidades técnicas en HTML y CSS como mi estilo visual personal, sirviendo como base para futuros desarrollos y ampliaciones del portfolio.
